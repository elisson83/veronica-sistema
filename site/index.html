Sistema Integrado

Conexão via QR Code

Os 3 painéis se conectam automaticamente com uma leitura de QR. Sem configuração manual, sem IPs, sem complicação.

Restaurante (PainelGest)
Gera QR Code no painel → Frota escaneia → Restaurante aparece automaticamente na frota
PainelFrota (Hub Central)
Gerencia restaurantes conectados + motoboys registrados. Gera QR individual para cada motoboy.
Motoboy (AppMotoboy)
Escaneia QR da frota → Conta criada já vinculada → Começa a receber entregas imediatamente
